Instructions on how to mine.

Step 1

Download the latest stable release from the below: https://github.com/tpruvot/cpuminer-multi/releases. Or you can visit http://mining-coins.com/

Step 2

Unzip the downloaded directory.

Step 3

Edit the file RUN-TimeTravel.bat and update the mining pool details with your own. It should look something like... cpuminer-gw64-corei7 -a timetravel -o stratum+tcp://38.242.240.186:3008 -u PVhUC99YkRviGQXmzxA4bNy3cZFDHb41pX -p c=MAC pause

Step 4

Double click the RUN-TimeTravel.bat to start mining.
